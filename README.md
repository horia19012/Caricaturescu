# Caricaturescu

**Caricaturescu** is a React-based e-commerce site for selling custom caricatures. Users can order their own caricature and choose a frame to personalize it. Orders are sent directly via **EmailJS**, making the process smooth and serverless.

## Features
- Order personalized caricatures
- Choose a frame
- Order form integrated with EmailJS
- Responsive React frontend

## Tech Stack
- **Frontend:** React
- **Email Integration:** EmailJS
